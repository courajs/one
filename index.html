What it's about

After leaving my last job I decided to commit some time to personal projects before I started the next job search. Unfortunately with no clear goal I was feeling a bit lost. So begins my Daily Sites challenge - publish a new web site live every single day.

Rules

  1. Every. Single. Day.
  2. It has to be alive on the internet by midnight. This means pushing something half done if necessary.
  3. Don't be lame. Every day has to have something unique. I don't have to start from scratch, but something has to be new. There's got to be a reason for the new site to exist.
  4. No regrets. Don't be a revisionist. These should stay alive forever(ish).
